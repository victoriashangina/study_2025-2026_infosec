# Контрольные вопросы

## Вопрос 1

**Какую информацию содержит учётная запись пользователя?**

Учётная запись содержит сведения для опознания пользователя, авторизации и учёта: идентификатор (login) и пароль.

## Вопрос 2

**Команды терминала:**

- *help* - справка
- *cd* - перемещение
- *ls* - просмотр содержимого
- *du* - объём каталога
- *mkdir/rmdir* - создать/удалить каталог
- *touch/rm* - создать/удалить файл
- *chmod* - права доступа
- *history* - история команд

## Вопрос 3

**Что такое файловая система?**

Файловая система определяет способ организации, хранения и именования данных.

Примеры:
- **FAT** - для флеш-накопителей
- **NTFS** - для Windows
- **Ext4** - для Linux

## Вопрос 4-5

**4. Как посмотреть подмонтированные ФС?**

Команда *df*

**5. Как удалить зависший процесс?**

Узнать PID через *ps*, затем *kill <PID>*
